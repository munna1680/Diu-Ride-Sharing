# 🚗 DIU Ride Sharing System

This is a Ride Sharing System project developed as part of academic coursework at **Daffodil International University (DIU)**.

---

## 📌 Project Description
The system allows users to:
- 🔐 Register and login  
- 🚕 Book rides  
- 📋 Manage ride requests  
- 👥 Handle driver and passenger interactions  

---

## 🛠️ Technologies Used
- 💻 Programming Language: **Java**  
- 🗄️ Database: **MySQL**  
- 🔧 Tools: **Git, GitHub**  

---

## 🚀 Features
- ✅ User Authentication System  
- ✅ Ride Booking System  
- ✅ Driver Management  
- ✅ Simple & User-Friendly Interface
